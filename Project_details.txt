Notre projet est rendu sous la forme d'un seul projet java sous eclipse. 
Pour lancer la pointeuse, il faut lancer le fichier "PointeuseController"
Pour lancer le programme principal, il faut lancer le fichier "Main_view"
Il n'y a aucune contraite concernant l'ordre de lancement des programmes. 
Malgré que les deux programmes utilisent une source commune de classes, les deux n'ont pas besoin de l'autre pour fonctionner.
Nous avons utilisé un modèle MVC pour les deux programmes, avec d'un côté le ou les controleurs qui s'occupe de gerer des données ou d'effectuer des actions (comme par exemple les listener) et de l'autre les vues qui s'occupe de l'organisation de l'affichage.
Nous avons organisé les données sérialisé dans des dossiers distint : 
- les données de la pointeuse sont dans le dossier "PointeuseData" et contiennent toutes les check in/out non envoyé par la pointeuse, ainsi qu'un fichier de configuration contenant la derniere adresse IP et le dernier port rentré.
- les données du programme principal sont dans le dossier "CoreData" et contiennent toutes les information de la companie, dont les département, employé, emploi du temps, check in/out, etc... ainsi qu'un fichier de configuration contenant uniquement le port utilisé pour la récéption des données.
